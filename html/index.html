Operations

  • PUB produto-alterado

    Esse topico representa produto criado

    Enviar informações de criação do produto.

    Operation IDpub-produto-alterado
    partitions
    3
    • #produto
    • #preço
    • #descricao

    Accepts the following message:

    ProdutoMensagem para o evento produto

    Evento para o produto

    • application/json
    object
    uid: produtoPayload
    idProduto
    integer
    >= 0

    Id Produto.

    descricao
    string

    Descrição produto.

    preco
    number

    Preço do produto.

    dataAtualizacao
    string

    Data da alteração/criação do produto.

    Additional properties are allowed.

    Examples

    • #1 Example - produto-criado-1s

      Exemplo de mensagem

      {
  "idProduto": 1,
  "descricao": "Celular bom",
  "preco": 1.99,
  "dataAtualizacao": "2022-05-23T13:06:42.375-0300"
}
